## ¿Qué es esto?

Web de contacto profesional desplegada vía **GitHub Pages** con dominio personalizado. Una sola página estática: sin frameworks, sin dependencias, sin build step.

---

## Stack

| Tecnología | Detalle |
|------------|--------|
| HTML + CSS + JS | Vanilla, un único fichero `index.html` |
| Fuente | Fira Code (Google Fonts) |
| Animación | Matrix canvas + scanline CSS |
| Hosting | GitHub Pages |
| Dominio | `theoffsecgirl.com` via CNAME |

---

## Estructura

